# Welltemplate
Complex fabric template mod generation. Currenly need to compile refactored verson

Allows generating template mods with entrypoints, registration, and several common datagen providers. [Tenreg](https://modrinth.com/mod/tenreg) support will be added before a public release
Template created from [The Example Fabric mod](https://github.com/FabricMC/fabric-example-mod), with large additions by me.
